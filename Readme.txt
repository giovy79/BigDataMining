Se encuentran dos carpetas Dataset y Notebooks

Dataset: Se hace referencia al set de datos tomado de movielens (https://grouplens.org/datasets/movielens/) archivo ml-25m.zip. 
Se intentó adicionar los archivos a este repositorio pero debido a la restricción de 100Mb que tiene Github y al tamaño de mas
de 1Gb de los archivos no fue posible. Se deben descargar principalmente los archivos ratings.csv y movies.csv.

Notebooks: Se observan dos archivos, EDA final.ipynb y Modelacion final.ipynb. En EDA final observamos los analisis de datos 
exploratorios realizados y en modelacion se observan los diferentes modelos y resultados obtenidos.
